Функционал синхроперевода позволит вам подключать в свои конференции переводчиков.

Назначить участника переводчиком может только модератор. Для этого необходимо пункт меню «Назначить переводчиком», как показано на рисунке ниже

После этого необходимо выбрать языки, с какого и на какой будет иди перевод:

  • Сверху указан язык, с которого будет идти перевод.
  • Снизу язык, на который будет идти перевод.

Если поставить галочку «двухсторонний перевод» переводчик будет переводить на оба выбранных языка.

В данной версии при выборе двухстороннего перевода, на обоих языках будет идти один и тот же перевод.

После того как нажмете «Да» участник будет назначен переводчиком.

Изображение переводчика пропадет из всех режимов отображения.
